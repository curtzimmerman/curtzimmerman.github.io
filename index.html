Web Scraper

Craigslist Scraper

A web scraper written using Ruby on Rails

Craigslist Scraper is a web scraper that was written to make searching nearby cities easier. Using Ruby on Rails and the Ruby Gem Nokogiri, it makes efficient http requests and parses the HTML that is received. Users can select their home city, and the app will automatically search the nearby cities for hits to their search query.

Technologies used:

Ruby on Rails, PostgreSQL, Bootstrap, Nokogiri

Facebook Clone

Odin Facebook Clone

A clone of the ubiquitous social network as part of The Odin Project

The final project for the Ruby on Rails course from The Odin Project. It is written to interface with Facebook's API so users can login using their Facebook information, or create a new account. Users can send friend requests, view profiles, make comments, and like other users' posts.

Technologies used:

Ruby on Rails, PostgreSQL, Devise, OmniAuth, Facebook API, Heroku Deployment
